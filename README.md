* RNN.ipynb - RNN-based text classifier
* LSTM.ipynb - LSTM-based text classifier
* src/rnn.py - RNN.ipynb as a py file
* src/lstm.py - LSTM.ipynb as a py file

## Folder: VITERBI
* vitberbi.ipynb - Implementation of Viterbi algorithm
